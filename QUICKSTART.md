# ⚡ Quick Start Guide

Get your Team Manager app running in **3 minutes**!

## 🏃‍♂️ Super Quick Setup

### Step 1: Install (2 minutes)
```bash
cd team-manager-app
npm install
```

### Step 2: Run (10 seconds)
```bash
npm run dev
```

### Step 3: Login (5 seconds)
Open browser to: `http://localhost:5173`

**Login with:**
- Email: `manager@example.com`
- Password: `password123`

## 🎯 Try These Features Now

### 1️⃣ Add Your First Player (30 seconds)
1. Click **"Players"** in sidebar
2. Click **"Add Player"** button
3. Fill in: Name, Age, Nationality, Height, Weight
4. Select positions (e.g., ST, CF)
5. Click **"Add Player"**

### 2️⃣ Schedule Your First Match (30 seconds)
1. Click **"Calendar"** in sidebar
2. Click **"Schedule Match"** button
3. Enter opponent name, date, location
4. Choose Home/Away
5. Select formation
6. Click **"Schedule Match"**

### 3️⃣ Record a Match Result (1 minute)
1. Go to **"Calendar"**
2. Find your match
3. Click **"Record Result"**
4. Enter scores (e.g., 3-1)
5. Select players who played
6. Add their stats (goals, assists, etc.)
7. Click **"Save Result"**

### 4️⃣ View Dashboard (10 seconds)
1. Click **"Dashboard"** in sidebar
2. See your team stats update automatically!

## 📱 What You Can Do

1. Go to **"Team Settings"**
2. Change team name
3. Pick your colors
4. Set preferred formation
5. Click **"Save Changes"**

## 💾 Your Data

All data is saved automatically in your browser!
- No account needed
- No internet required
- Data persists forever (until you clear browser data)

## 🆘 Having Issues?

### Can't Start?
```bash
# Make sure Node.js is installed
node --version  # Should show v18 or higher

# Reinstall dependencies
rm -rf node_modules
npm install
```

### Port Already in Use?
```bash
# Use different port
npm run dev -- --port 3000
```
